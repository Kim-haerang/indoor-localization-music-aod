# 공개 범위

이 저장소는 학부연구생 활동의 공개 가능한 요약본입니다.

## 공개하는 것

- 연구 주제와 접근 방법
- 개인 학습 내용
- 대표 결과 이미지
- 실험 흐름 요약
- 공개 가능한 수준의 기술 설명

## 공개하지 않는 것

- 연구실 내부 원본 데이터
- 대용량 `.mat`, `.h5`, `.csv` 파일
- 지도교수 또는 연구실 소유 문서
- 미공개 연구 결과
- 재현 가능한 전체 학습 데이터셋

## 이유

학부연구생 활동은 개인 포트폴리오로 설명할 수 있지만, 연구실 자료와 미공개 결과는 소유권과 공개 권한을 확인해야 합니다. 따라서 이 저장소는 개인이 수행한 역할과 학습 내용을 설명하는 공개용 문서 중심으로 구성합니다.
