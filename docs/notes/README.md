# 연구 노트

MUSIC 기반 실내 위치추정 가능성을 검토하면서 정리한 공개용 연구 노트입니다.

## 핵심 키워드

- 실내 위치추정
- AoD / DoA
- MUSIC 알고리즘
- 안테나 배열
- SNR 변화
- phase wrapping / unwrapping
- CFO 보정
- CNN 기반 denoising

## 정리 기준

연구실 내부 자료나 원본 데이터를 그대로 공개하지 않고, 개인 학습과 실험 흐름을 설명할 수 있는 범위만 정리합니다.
